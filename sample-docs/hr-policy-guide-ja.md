# 人事規程ガイド

## 1. 休暇制度

### 1.1 年次有給休暇
- **付与基準**: 入社1年目 15日、2年目以降毎年1日追加（最大25日）
- **使用期限**: 発生年度＋1年以内に使用（繰越不可）
- **申請方法**: Workday > 休暇申請 > 承認
- **承認者**: 直属上司（3日以上連続休暇は部門長）
- **最小申請単位**: 0.5日（半休）可
- **未使用時**: 年末精算時に手当へ転換（1日 = 平均賃金1日分）

### 1.2 病気休暇
- **年間上限**: 最大10日
- **3日以上連続**: 医師の診断書提出必須
- **長期病気休暇**: 30日超過時は休職手続きへ移行
- **有給可否**: 5日まで有給、6日目以降は無給（または年休消化選択）

### 1.3 慶弔休暇
| 事由 | 日数 | 備考 |
|---|---|---|
| 本人結婚 | 5日 | 招待状のコピー提出 |
| 子の結婚 | 2日 | 招待状のコピー提出 |
| 親・配偶者死亡 | 5日 | 死亡診断書 |
| 祖父母・兄弟姉妹死亡 | 3日 | 死亡診断書 |
| 配偶者出産 | 10日 | 出生証明書 |
| 本人出産 | 90日 | 医療証明 |

### 1.4 リフレッシュ休暇
- **勤続5年**: 5日 + 補助金 5万円
- **勤続10年**: 10日 + 補助金 10万円
- **勤続20年**: 15日 + 補助金 20万円
- **使用期限**: 勤続到達日から1年以内

### 1.5 特別休暇
- **誕生日休暇**: 年1日（誕生日週内に自由使用）
- **ワクチン休暇**: 接種当日 + 翌日（必要時）
- **介護休暇**: 年5日（無給）

## 2. 勤務時間

### 2.1 通常勤務
- **勤務時間**: 09:00 ~ 18:00（計8時間、昼休み1時間除く）
- **昼休み**: 12:00 ~ 13:00（部署別調整可）
- **休憩時間**: 午前・午後 各10分（自由）

### 2.2 コアタイム（在席必須）
- **時間帯**: 10:00 ~ 16:00
- **必須事項**: 会議出席、即時応答可能な状態維持
- **例外**: 外出/出張/事前承認された一時外出

### 2.3 フレックス勤務
- **時差出退勤**: 07:00 ~ 10:00 の間で自由出勤
- **裁量労働**: 1ヶ月平均週40時間（管理者承認必要）
- **申請方法**: Workday > フレックス勤務申請

### 2.4 リモートワーク
- **回数**: 週2回まで（部署別異なる）
- **申請**: 前営業日17時までに Workday
- **要件**: VPN接続、コアタイム在席、会社支給機器使用
- **手当**: 月3,000円（回線・電気補助）

### 2.5 残業・休日勤務
- **事前承認必須**: 部門長承認
- **手当**: 残業時 時給1.5倍、休日は2倍
- **代休**: 残業8時間につき1日付与（選択）
- **上限**: 月45時間（法定上限）

## 3. 出張及び経費

### 3.1 国内出張
| 項目 | 上限 | 備考 |
|---|---|---|
| 一日食事代 | 5,000円 | 領収書添付 |
| 宿泊（東京外） | 15,000円/泊 | ビジネスホテル基準 |
| 宿泊（東京） | 18,000円/泊 | |
| 交通 | 実費 | 新幹線指定席、航空エコノミー |
| 車両使用 | km当たり 50円 | 給油領収書 |

### 3.2 海外出張
- **事前承認**: 部門長 + 経営支援本部
- **航空便**: 8時間以上のフライトはビジネス可（管理者以上）
- **宿泊**: 都市別上限（ソウル 25,000円/泊、ニューヨーク 30,000円/泊）
- **食事代**: 一日 USD 70~100（地域別）
- **ビザ/予防接種**: 会社負担

### 3.3 精算手続き
1. **事前承認**: 出張申請書承認（出発5日前）
2. **法人カード**: 可能な場合使用（月上限30万円）
3. **領収書**: 使用日から30日以内に提出
4. **精算**: 毎月25日締め切り
5. **承認**: 本人 → 課長 → 財務部（3段階）

### 3.4 事前承認必要ケース
- 単一支出 5,000円以上
- 出張（国内・海外共に）
- 法人カード未使用時
- 外部会食・接待
- 学会・セミナー参加

## 4. 評価及び昇進

### 4.1 評価制度
- **周期**: 年2回（上半期7月、下半期1月）
- **方式**: 目標管理（MBO）+ 360度フィードバック
- **評価者**: 本人（自己）+ 上司 + 同僚3名 + 部下（該当時）
- **等級**: S（5%）/ A（20%）/ B（50%）/ C（20%）/ D（5%）

### 4.2 360度フィードバック
- 匿名で実施
- 評価者プール: 本人選定後HRが確定
- 項目: リーダーシップ、協働、専門性、成果（4領域）

### 4.3 昇進審査
- **時期**: 毎年12月
- **要件**: 等級別最低勤続 + 評価等級 + 社内試験（該当時）
- **結果発表**: 翌年1月1日付
- **異議申立**: 結果発表後14日以内

### 4.4 等級体系
| 等級 | 最低勤続 | 備考 |
|---|---|---|
| 社員 | - | 入社時 |
| 主任 | 4年 | |
| 課長 | 4年 | 評価A以上1回必須 |
| 次長 | 4年 | |
| 部長 | 5年 | 役員候補 |
| 役員 | - | 別途審査 |

- **残業食事代**: 1,000円支援（20:00以降残留時）
- **会食**: 部署当たり四半期3万円

### 5.5 その他

## 6. 入退社手続き

### 6.1 入社
- **準備物**: 身分証、通帳コピー、住民票、学歴証明書、職歴証明書
- **初週スケジュール**:
  - Day 1: 入社処理、ノートPC受領、セキュリティ教育
  - Day 2~3: 部署紹介、メンター割当
  - Day 4~5: 職務OJT開始
- **試用期間**: 3ヶ月（評価後正社員転換）

### 6.2 部署異動
- **申請時期**: 年2回（4月、10月）
- **公募掲載**: 社内ポータル > 人事 > 社内公募
- **要件**: 現部署1年以上勤続、評価B以上

### 6.3 退社
- **辞表提出**: 退社30日前（書面）
- **引継ぎ**: 最低2週間
- **退職金**: 1年以上勤続時（平均賃金30日分 × 勤続年数）
- **資産返納**: 最終日17時まで（ノートPC、IDカード、法人カード）
- **退職面談**: HR + 直属上司

### 6.4 休職
- **育児休職**: 子1人当たり最大1年（男女共）
- **学業休職**: 最大2年（関連学科）
- **病気休職**: 医師所見書 + 最大1年
- **復職**: 休職終了30日前に申請

## 10. よくある質問（FAQ）

### Q1. 年休を使いきれなかったらどうなりますか？
- A: 発生年度 + 1年以内に使用する必要があります。未使用分は年末に手当へ自動転換されます（1日 = 平均賃金1日分）。

### Q2. リモートワークは自由にできますか？
- A: 週2回まで可能で、前営業日17時までにWorkdayで申請する必要があります。コアタイム（10-16時）には必ず在席が必要です。

### Q3. 出張時の航空機座席等級は？
- A: 等級無関係に8時間未満はエコノミーです。8時間以上のフライトは管理者以上に限りビジネス可能です。

### Q4. 評価等級はどう決まりますか？
- A: 自己評価 + 上司評価 + 360度フィードバックを総合します。部署別に正規分布（S 5%、A 20%、B 50%、C 20%、D 5%）に従います。

### Q5. 両親の還暦祝いは慶弔休暇がありますか？
- A: いいえ。本ガイドの慶弔休暇には含まれていません。必要時は年休をご利用ください。

### Q6. 新入社員試用期間中に退社した場合は？
- A: 試用期間中も退職金以外の一般手続きは同じです。ただし、入社時に受け取ったサインオンボーナスがあれば返還義務が発生する可能性があります。

### Q7. 育児休職は男性も可能ですか？
- A: はい、男女共に子1人当たり最大1年まで使用可能です。出産休暇（配偶者出産時10日）とは別です。

### Q8. 会社社宅の申請資格は？
- A: 新婚5年以内（婚姻届基準）または地方発令者（3年）です。申請は人事部に直接お問い合わせください。
